## Getting Started

```bash
# Clone the repository
git clone https://github.com/yourname/dcr.git
cd dcr

# Install dependencies
npm install

# Run local dev server
npm run dev

# Compile contracts
npx hardhat compile

# Deploy contract to Somnia testnet
npx hardhat run scripts/deploy.cjs --network somnia
```
